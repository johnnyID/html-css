Utilizando ID

id é uma forma de configuração individual. Ou seja, podemos ter uma personalização única para cada tag, sobrepondo uma configuração universal. É recomendado utilizar apenas 1 id em tags. Uso de id.

Aqui está utilizando uma configuração universal para todos os parágrafos, ou seja, não importa quantos parágrafos utilizar, todos estarão com cores verdes. Sem uso de id.
