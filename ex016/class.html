Conceito de Class

Class é uma forma de organizar uma configuração/personalização que poderá ser utilizada para diversas tags,

Aplicando Conceito

Para utilizar uma class, antes temos de criar a class nas CSS para então, incorporar no HTML5, vejamos: 

            
        .sem-serifa{
            font-family: Arial, Helvetica, sans-serif;
        }
        .azul{
            color: blue;
        }
        .verde{
            font-weight: bold;
        }

. em CSS = class em HTML5
