Pseudo-Element

Conceito

Diferente da pseudo-class que se permite estilizar de acordo com o estado do elemento, os pseudo-elementos permite estilizar uma parte específica de um elemento selecionado, por exemplo, adicionar algum texto antes ou após um link ou parágrafo, sendo tudo apenas visual, não sendo conteúdo semântico. Vejamos: 

            
        h2::before{
            content: "> ";
        }
        h2::after{
            content: " <";
        }

Aplicando Conceito

Como a demonstração acima, ao aplicar, todos os títulos de hierarquia 2 (h2) terão duas setas apontadas para eles, uma antes e uma depois.
