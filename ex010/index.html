Estrutura para Associar Links

Estrutura básica

Utilizamos a tag <a></a> com o parâmetro href="" para associarmos o conteúdo à um site externo ou interno. O parâmetro href="" é onde adicionamos o caminho/url do site para qual estamos direcionando o usuário, sendo um site externo ou interno, sobrepondo a página do mesmo. Vejamos. 

            
    <a href=""></a> <- Corpo da estrutura

Parâmetros

  • href=""

    • Utilizado para associarmos o conteúdo à um site externo ou interno para o qual estamos direcionando o usuário.

    • Link Externo

      • Para o link externo, passamos como parâmetro, a url do site, Vejamos. 

                          
    <a href="https://ww.youtube.com"> Clique aqui </a>
    • Link Interno

      • Para o link Interno, passamos como o parâmetro o caminho do diretório onde está o arquivo de outra página do mesmo site, Vejamos. 

                          
    <a href="pag002.html"> Clique aqui </a>

      O código acima utiliza de uma estrutura que o caminho do arquivo está em uma mesma pasta do código da página principal do site, ou seja, apenas adicionamos o arquivo da página que queremos, tanto para irmos para a próxima página quanto para voltarmos de página. 

                          
    <a href="arpa/pag003.html"> Clique aqui </a>
                    
                
                          
    <a href="../pag003.html"> CLique aqui </a>

      O código acima utiliza de uma estrutura onde o objetivo é voltar de página(página principal do site, por exemplo). O caminho do arquivo está em uma pasta diferente do código da página principal do site, ou seja, é preciso utilizar de ../ para voltar uma hierarquia do diretório.

      Obs: Ao adicionarmos o caminho do diretório, utilizamos do sistema utilizado no Linux, onde a hierarquia de diretório existe.
      ./ -> Significa que está em uma mesma pasta.
      ../ -> Volta 1 pasta.
      ../../ -> Volta 2 pastas.
      ../../../ -> Volta 3 pastas e por assim vai.
      Ou seja, tudo vai depender da hierarquia de diretório onde o arquivo se encontra.

  • target=""

      • Utilizado para especificar se o usuário vai abrir uma nova janela, ou se vai sobrepor a página em que se encontra por outra.

    • target="_blank"
      •                     
<a href="https://www.youtube.com" target="_blank"> Clique aqui </a>

      Será aberto uma nova janela para o site que o usuário deseja acessar.

    • target="_self" 
      •                     
<a href="https://www.youtube.com" target="_self"> Clique aqui </a>
  • rel=""

      • Utilizado para forma de busca do google, indica se o link é externo, se o link vai para uma próxima página ou para uma anterior.

    • rel="external" 
      •                     
<a href="https://www.youtube.com" rel="external"> Clique aqui </a>

      Indicando que o link é externo, ou seja, vai para algum outro site.

    • re="next" 
      •                     
<a href="pag002.html" rel="next"> Clique aqui </a>

      Indicando que o link vai redirecionar o usuário para uma próxima página do mesmo site.

    • re="prev" / re="previous" 
      •                     
<a href="../pag002.html" rel="prev"> Clique aqui </a>
