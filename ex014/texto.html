Alinhamento de Texto

Centralizando Texto

Antes do HTML5, para centralizarmos um texto, utilizávamos da tag <center>, contudo viera se tornar obsoleta, deixando esta função para as CSS. Ou seja, para centralizarmos textos, utilizamos da CSS com a declaração da seguinte forma:

        
    seletor{
        text-align: center;
    }

Texto à Esquerda

Para direcionarmos o texto à esquerda, utilizamos a declaração da seguinte forma: 

        
    seletor{
        text-align: left;
    }

Texto à Direita

Para direcionarmos o texto à direita, utilizamos a declaração da seguinte forma: 

        
    seletor{
        text-align: right;
    }

Texto Justificado

Texto justificado alinha o texto nas extremidades da tela, sendo dinâmico de acordo com a largura e tamanho da própria tela. Utilizamos a declaração da seguinte forma: 

        
    seletor{
        text-align: justify;
    }
